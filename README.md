# 🧬 SafeCell

**SafeCell** is an AI-powered malaria diagnosis dashboard built to help healthcare workers in rural areas detect malaria from blood smear images, analyze regional infection trends, and collect community feedback — all within a responsive and accessible user interface.

---


## 🚀 Features

- 🧠 **Malaria Detection**  
  Upload and analyze blood smear images using an AI-powered detection model.

- 📊 **Detection Statistics**  
  Track trends and monitor infection patterns over time using interactive visualizations (powered by Chart.js).

- 🗺️ **Location-Based Insights**  
  Map integration for understanding spatial trends in malaria cases.

- 📱 **Responsive Design**  
  Optimized for desktop, tablet, and mobile screens.

- 🌗 **Dark Mode Support**  
  Easily toggle between light and dark themes for user comfort.

- 🌍 **Multilingual Interface**  
  Designed for accessibility in multiple languages.

- 💬 **Community Feedback**  
  Collect feedback from users and healthcare workers to improve system effectiveness.

- 🔐 **Login Functionality (Mock Session)**  
  Simple login interface with placeholder authentication to simulate real usage.
